### **Phần 1: Thông số chi tiết cho Powerpoint**

Để tạo một infographic kích thước 2m x 1m với chất lượng in tốt bằng Powerpoint, bạn không thể đặt kích thước slide là 200cm x 100cm vì Powerpoint có giới hạn kích thước tối đa là 142.24 cm.

Do đó, phương pháp đúng là làm việc trên một slide có **tỷ lệ chính xác (2:1)** và sau đó **xuất file ở độ phân giải cao (DPI)**.

Dưới đây là các thông số chi tiết bạn cần thiết lập:

#### **1. Page Layout (Thiết lập trang)**

1.  Mở Powerpoint, vào tab **Design** (Thiết kế).
2.  Chọn **Slide Size** (Kích thước trang chiếu) -> **Custom Slide Size...** (Tùy chỉnh kích thước trang chiếu...).
3.  Trong hộp thoại hiện ra, thiết lập các thông số sau:
    *   **Width** (Rộng): **100 cm** (hoặc 50 cm, miễn là giữ tỷ lệ 2:1 và đủ lớn để làm việc).
    *   **Height** (Cao): **50 cm** (tương ứng với 100cm ở trên).
    *   **Orientation** (Hướng): **Landscape** (Ngang).

*Lý do:* Chúng ta làm việc ở tỷ lệ 1/2 (100x50 cm) so với kích thước thật (200x100 cm). Khi xuất file với DPI là 200, ảnh sẽ có kích thước pixel chính xác như yêu cầu.
*Tính toán:* (100 cm / 2.54 cm/inch) * 200 DPI ≈ 7874 pixels.

#### **2. Font Size (Kích cỡ chữ) - Font Roboto**

Với kích thước in lớn, cỡ chữ cần phải đủ lớn để người xem có thể đọc được từ khoảng cách 1-2 mét. Dưới đây là gợi ý về hệ thống phân cấp cỡ chữ sử dụng font Roboto:

*   **Tiêu đề chính (DỮ LIỆU CỦA BẠN - QUYỀN CỦA BẠN):**
    *   **Font Size:** 130 - 160 pt
    *   **Font Weight:** Roboto Black hoặc Roboto Bold
    *   *Ghi chú:* Cỡ chữ lớn nhất, thu hút sự chú ý đầu tiên.

*   **Tiêu đề phụ (Luật Bảo vệ dữ liệu cá nhân 2025...):**
    *   **Font Size:** 80 - 100 pt
    *   **Font Weight:** Roboto Bold hoặc Roboto Medium

*   **Tiêu đề các mục lớn (Mục 1, 2, 3, 4):**
    *   **Font Size:** 72 - 90 pt
    *   **Font Weight:** Roboto Bold
    *   *Ghi chú:* Dùng màu sắc hoặc icon để phân biệt các mục.

*   **Tiêu đề con trong các mục (VD: IM LẶNG KHÔNG PHẢI LÀ ĐỒNG Ý!):**
    *   **Font Size:** 54 - 66 pt
    *   **Font Weight:** Roboto Bold hoặc Roboto Medium Italic
    *   *Ghi chú:* Phải nổi bật hơn nội dung thường nhưng nhỏ hơn tiêu đề mục.

*   **Nội dung chính (Các gạch đầu dòng, giải thích):**
    *   **Font Size:** 36 - 44 pt
    *   **Font Weight:** Roboto Regular
    *   *Ghi chú:* Đảm bảo dễ đọc, khoảng cách dòng (line spacing) nên đặt ở mức 1.2 đến 1.5 để thoáng mắt.

*   **Văn bản nhỏ, ghi chú (Ví dụ, chú thích, nguồn...):**
    *   **Font Size:** 24 - 28 pt
    *   **Font Weight:** Roboto Light hoặc Roboto Regular

#### **3. Cách xuất file chất lượng cao**

Powerpoint mặc định xuất ảnh ở độ phân giải thấp (96 DPI). Để đạt được 7874x3937 pixels, bạn cần xuất ở 200 DPI.

1.  Vào **File** -> **Export** (Xuất).
2.  Chọn **Change File Type** (Thay đổi loại tệp).
3.  Chọn **PNG** hoặc **JPEG**.
4.  Khi bấm **Save**, Powerpoint sẽ hỏi bạn muốn xuất slide nào. Chọn **Just This One** (Chỉ trang này).
5.  **Quan trọng:** Powerpoint sẽ không hỏi độ phân giải. Để tăng độ phân giải mặc định, bạn cần chỉnh sửa Registry (chỉ dành cho Windows).
    *   Mở Registry Editor (gõ `regedit` vào ô tìm kiếm).
    *   Điều hướng đến `HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\PowerPoint\Options` (số `16.0` có thể thay đổi tùy phiên bản Office).
    *   Tạo một giá trị **DWORD (32-bit) Value** mới, đặt tên là `ExportBitmapResolution`.
    *   Click đúp vào nó, chọn hệ **Decimal** (Thập phân) và nhập giá trị **200**.
    *   Khởi động lại Powerpoint và xuất file. Giờ đây file ảnh của bạn sẽ có chất lượng cao đúng yêu cầu.

### **Phần 2: Cập nhật code Python**

Tôi đã đọc kỹ code của bạn và nội dung mới về Luật Bảo vệ dữ liệu cá nhân. Dưới đây là phiên bản code đã được cập nhật hoàn toàn để phù hợp với đề bài mới.

**Những thay đổi chính:**

1.  **Chủ đề & Nội dung:** Toàn bộ văn bản, tiêu đề, các mục đã được thay thế bằng nội dung về Luật Bảo vệ dữ liệu cá nhân 2025.
2.  **Bố cục (Layout):**
    *   Bố cục được thiết kế lại để phù hợp với nội dung luật: Tiêu đề -> 4 Khái niệm cốt lõi -> 3 Cột chính (Quy định nổi bật, Điểm mới, Quyền & Nghĩa vụ) -> Footer.
    *   Loại bỏ phần "Thống kê nổi bật" không phù hợp.
3.  **Màu sắc & Phong cách:**
    *   Thay đổi bảng màu từ đỏ-vàng (chủ đề quốc khánh) sang tông màu xanh dương - xanh lá - xám (chủ đề công nghệ, pháp luật, an toàn), tạo cảm giác chuyên nghiệp, tin cậy.
    *   Sử dụng các icon đơn giản (emoji) để minh họa cho dễ hiểu.
4.  **Prompt tạo ảnh AI:**
    *   Các prompt đã được viết lại hoàn toàn bằng tiếng Việt, tập trung vào chủ đề an toàn dữ liệu, công nghệ, và pháp luật để tạo ra các hình ảnh nền và minh họa phù hợp.
5.  **Fonts:** Đã đổi từ Arial sang **Roboto** như yêu cầu. Bạn cần đảm bảo đã cài đặt font Roboto trên máy của mình.
6.  **Tên hàm và biến:** Đã được đổi lại cho rõ ràng và phù hợp với chủ đề mới (ví dụ: `create_infographic_luat_bvdl`).

def generate_image_with_ai(prompt_vietnamese, output_filename, size="1536x1024"):
    """Tạo ảnh bằng AI với prompt tiếng Việt"""
    print(f"   ⏳ Đang tạo ảnh cho: {output_filename.split('/')[-1]}...")
    try:
        response = CLIENT.images.generate(
            model="imagen-4",
            prompt=prompt_vietnamese,
            n=1,
            size=size
        )
        if response.data and len(response.data) > 0:
            image_data = response.data[0]
            if hasattr(image_data, 'b64_json') and image_data.b64_json:
                img = Image.open(BytesIO(base64.b64decode(image_data.b64_json)))
            elif hasattr(image_data, 'url') and image_data.url:
                img_response = requests.get(image_data.url)
                img = Image.open(BytesIO(img_response.content))
            
            img.save(output_filename)
            print(f"   ✅ Đã lưu ảnh: {output_filename}")
            return img
    except Exception as e:
        print(f"   ❌ Lỗi khi tạo ảnh: {e}")
        return None
    return None

def draw_shadow_box(draw, xy, radius, fill_color, shadow_offset=15, shadow_color='#00000030'):
    """Vẽ một hộp có bóng đổ"""
    x1, y1, x2, y2 = xy
    shadow_xy = (x1 + shadow_offset, y1 + shadow_offset, x2 + shadow_offset, y2 + shadow_offset)
    draw.rounded_rectangle(shadow_xy, radius, fill=shadow_color)
    draw.rounded_rectangle(xy, radius, fill=fill_color)

def wrap_text(text, font, max_width, draw):
    """Ngắt dòng văn bản để vừa với chiều rộng cho trước"""
    lines = []
    if not text:
        return lines
    words = text.split()
    while words:
        line = ''
        while words and draw.textbbox((0, 0), line + words[0], font=font)[2] <= max_width:
            line += (words.pop(0) + ' ')
        lines.append(line.strip())
    return lines

    print(f"✅ Đã lưu PNG: {output_png}")

    output_jpg = f"{output_filename}.jpg"
    # Chuyển sang RGB trước khi lưu JPEG
    img.convert('RGB').save(output_jpg, 'JPEG', dpi=(DPI, DPI), quality=95)
    print(f"✅ Đã lưu JPG: {output_jpg}")

    print("✨ HOÀN THÀNH INFOGRAPHIC!")

    print(f"📐 Kích thước in tương đương: 2m x 1m")
    print("=" * 100)
